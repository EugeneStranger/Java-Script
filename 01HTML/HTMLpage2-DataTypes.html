Типы данных в языке C++

Переменная – это именованная область памяти, содержимое которой может изменяться в процессе выполнения программы. Прежде, чем использовать переменную, её нужно объявить. Для объявления переменной нужно указать её тип и имя следующим образом:

		
            
            type name;

type (тип переменной), определяет тип переменной:

  1. Сколько памяти переменная будет занимать;
  2. Какие значения она будет принимать;
  3. Какие операции над ней можно выполнять.

Все типы данных в языке C++ можно разделить на следующие категории:

  1. Логические типы. Включает в себя всего один тип данных – bool (Boolean). Переменная типа bool занимает 1 Байт памяти и принимает одно из двух значений: true или false true хранится в памяти как 1, а false как 0. На самом деле 0 – это false, а true – это всё, что НЕ 0.
  2. Символьные типы. Включает в себя всего один тип данных – char. Переменная типа char занимает 1 Байт памяти и содержит один единственный символ в кодировке ASCII. Всего в кодировке ASCII 256 символов, поэтому переменная типа char хранит один из 256 символов.
  3. Числовые типы. Делятся на две категории:
    • Целочисленные. Могут быть знаковыми (Signed) и беззнаковыми (Unsigned). Signed могут хранить положительные и отрицательные значения. Unsigned могут хрантить только положительные значения.
      К целочисленным относятся следующие типы данных:
      1. Short. Переменная типа short занимает 2 Байтa памяти и может принимать следующие значения:
        • unsigned short в диапазоне: 0...65535 (0...216-1);
        • signed short в диапазоне: -32768...32767 (-215...215-1).
      2. Long. Переменная типа long занимает 4 Байтa памяти и может принимать следующие значения:
        • unsigned long в диапазоне: 0...4294967295 (0...232-1);
        • signed long в диапазоне: -2147483648...2147483647 (-231...231-1).
      3. Int – платформенно-зависимый тип данных. Его размер зависит от типа процессора, операционной системы и среды разработки. Но в любом случае размер должен быть больше или равен размеру типа short и меньше или равен размеру типа long. В Microsoft Visual Studio под Windows, int всегда занимает 4 Байта памяти и полностью совпадает с long.
      4. Long long. Переменная типа long long занимает 8 Байт памяти и может принимать следующие значения:
        • unsigned long long в диапазоне: 0...4294967295 (0...264-1);
        • signed long long в диапазоне: -2147483648...2147483647 (-263...263-1).
    • Вещественные (числа с плавающей точкой). К вещественным относятся следующие типы данных:
      1. Float. Переменная типа float занимает 4 Байтa памяти и может принимать следующие значения в диапазоне: +(-)1.17549e-38...3.40282e+38.
        Переменная типа float способна хранить 38 знаков после запятой с точностью до 7 знаков.
      2. Double. Переменная типа double занимает 8 Байт памяти и может принимать следующие значения в диапазоне: +(-)2.22507e-308...1.79769e+308.
        Переменная типа double способна хранить 308 знаков после запятой с точностью до 15 знаков.
